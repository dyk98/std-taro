# 规范化Taro编写架构std-taro
一个规范化编写的taro1.2.8demo示例

使用京东凹凸造的多端轮子Taro v1.2.8 https://github.com/NervJS/taro

全局安装taro/cli1.2.8 

npm: `npm install -g @tarojs/cli@1.2.8`

cnpm: `cnpm install -g @tarojs/cli@1.2.8`

yarn: `yarn global add @tarojs/cli@1.2.8`
